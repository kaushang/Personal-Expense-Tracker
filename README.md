# Personal Expense Tracker API

A Node.js backend for tracking personal expenses.

## Features

- RESTful API
- MongoDB integration
- Express server

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on the template and add your MongoDB URI.

## Usage

### Development

```bash
npm run dev
```

### Production

```bash
npm start
```

## Folder Structure

- `src/models`: Database models
- `src/routes`: API routes
- `src/controllers`: Route logic
- `src/middleware`: Custom middleware
- `src/utils`: Helper functions
- `src/config`: Configuration files
